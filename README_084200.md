# Responsive Portfolio Site

## Overview
A mobile-first, responsive portfolio website showcasing my data operations projects.

## Features
- Semantic HTML5 structure
- CSS3 Flexbox and Grid layout
- Mobile-first responsive design
- Cross-browser compatibility (Chrome/Firefox)
- Hover animations on project cards

## Tech Stack
- HTML5
- CSS3
- Git/GitHub for version control

## How to View
1. Clone the repository
2. Open `index.html` in any browser
3. Resize window to see responsive behavior

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
